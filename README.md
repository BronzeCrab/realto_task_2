# realto_task_2

## Как запустить (на примере голой ubuntu 14.04):

ставим гит

-  `sudo apt-get install git`

клонируем этот репо

-  `git clone https://github.com/BronzeCrab/realto_task_2.git`

пип

-  `wget https://bootstrap.pypa.io/get-pip.py`
-  `python get-pip.py`

зависимости

-  `pip install -r requirements.txt'`
